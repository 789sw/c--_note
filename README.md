## 質數表
可以把質數的倍數都去掉
```
int p[1045]={0}
    for(int i=2;i<p[1045];i++){
        if(p[i]==0){
            for(int j=i+i;j<1045;j+=i)//將i的倍數設定成不是質數
            { 
                p[j]=1;
            }
        }
    }
```

## 費氏數列遞迴版(較久)
```
void fib(int n){
    if(n==0||n==1) return 1;
    else return fib(n-1)+fib(n-2);
}
```
